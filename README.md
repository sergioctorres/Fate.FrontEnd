# Fate.FrontEnd
FrontEnd em AngularJS para realização de CRUD através de API.

### Instalação


```sh

- Abrir a Solução Fate.FrontEnd;

- Digitar CTRL+SHIFT+B;

- Apertar F5 para executar a aplicação;

```



### Tecnologias

* AngularJS

* Bootstrap 3



### Ferramentas

* Visual Studio Community 2017
